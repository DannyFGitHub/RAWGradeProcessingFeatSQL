A UI on JAVAFX for Interacting with SQL Database

# Dependencies

mysql-connector-java-8.0.17.jar

---
#### NOTES For Once Program is Launched

When creating the cable:
- The table created will be called GradeProcessing
             
The credentials given MUST be a user with the ability to make changes.
The database must be created before running the program.    

SEARCHING RECORDS:
Type in the search for: field the full or partial search terms you would like to search for.
From the dropdown chose the field you would like to search in.
